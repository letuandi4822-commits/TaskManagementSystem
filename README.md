# Task Management System

A web-based task management system built with ASP.NET Core MVC.

## Features

- User Authentication
- Role-based Authorization (Admin/User)
- Project Management
- Task Assignment
- Email Notification
- File Upload
- Dashboard with Charts
- Activity Logs
- Notification Center

## Technologies

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server LocalDB
- Bootstrap 5
- Chart.js
- MailKit
